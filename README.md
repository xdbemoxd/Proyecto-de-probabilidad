# Proyecto-de-probabilidad
Proyecto donde se realiza un analisis de un conjunto de datos provenientes de un csv

# Primer paso

este es el primer paso para crear un entorno virtal, tanto en linux como en windows

```python -m venv env```

si tiene windows

```.\env\Scripts\activate```

si tiene linux 

```source env/bin/activate```

para despues continuar con la instalacion de los paquetes 

```pip install -r requirements.txt```

ya una vez completado esos pasos, puede compilar el archivo, usando el siguiente comando 

```python main.py```


# Asegurese siempre de estar en la carpeta correspondiente
